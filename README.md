# LLM Stylometric Fingerprinting

## Project Title

**The Hidden Signature of Machines: Stylometric Fingerprinting of Large Language Models Across Genres**

## Objective

This project investigates whether different Large Language Models have detectable writing styles. It uses controlled prompts across multiple genres, retrieves model responses through APIs, cleans the generated outputs, extracts stylometric features, and evaluates whether model identity can be detected through statistical analysis, semantic mapping, and classification.

## Main Research Question

Can stylometric features distinguish texts generated by different Large Language Models?
